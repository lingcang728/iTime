## 验收记录

- 真实 KeyStats 日数据在桌面 UI 与源文件逐项比对一致；测试时读取到 30 天历史、Top 键位与组合键，未修改源文件。
- iTime 活动采集器在桌面运行时持续写入脱敏区间，记录中不含标题、完整路径、PID 或键入内容。
- 自启动开关完成关闭、开启、再关闭的真实注册表往返；iTime 最终为关闭，KeyStats 启动项始终存在且值未变化。
- 原型关键页结构 SSIM：主页 0.916、AI 代理 0.939、周报 0.904；主要锚点偏差为 0、0、1 CSS px。
- Tauri Windows FFI 使完整 Miri 运行不可用；本轮采用原生 Rust 测试、clippy、fmt、真实图标提取与桌面运行验证补足该边界。

## 真实数据说明

接入之前没有完整的应用前台与 AI 执行历史，因此不会用 mock 补齐。KeyStats 仅能提供真实的日级键鼠聚合；iTime 自有应用时间线从本次启用后开始积累。AI 工具进程前台区间会标为估算，工具日志正文与窗口标题默认不读取。
